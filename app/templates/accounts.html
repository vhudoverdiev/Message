Этап 4: подключение сообщества через unified-архитектуру

  1. Получите community token c правами на сообщения сообщества.
  2. Добавьте сообщество: название, ID сообщества и токен.
  3. Нажмите Проверить токен — система выполнит проверку через groups.getById.
  4. Синхронизация диалогов выполняется автоматически каждую секунду через messages.getConversations.

Личный VK user token в этом разделе не используется.
