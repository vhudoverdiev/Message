# Flow Builder 2.0

Готовый проект версии 2.0: frontend + backend на FastAPI.

## Что это такое

Это уже не просто макет, а небольшой рабочий проект:

- красивый интерфейс в стиле Flow Builder
- backend на FastAPI
- загрузка текущего сценария с сервера
- добавление шагов
- удаление шагов
- сохранение сценария в JSON-файл
- публикация сценария
- сброс к базовому шаблону

## Как запустить

### 1. Распакуй архив

### 2. Открой терминал в папке проекта

### 3. Установи зависимости

```bash
pip install -r requirements.txt
```

### 4. Запусти сервер

```bash
python app.py
```

(Альтернатива: `uvicorn main:app --app-dir app --reload`)

### 5. Открой в браузере

```text
http://127.0.0.1:8000
```

## Что делает backend

Backend:
- отдает frontend
- хранит сценарий в `app/data/flow.json`
- умеет:
  - получить сценарий
  - сохранить сценарий
  - сбросить сценарий

## API

### Получить сценарий
`GET /api/flow`

### Сохранить сценарий
`POST /api/flow`

### Сбросить сценарий
`POST /api/flow/reset`

### Проверка сервера
`GET /api/health`
